# 🌐 Ibtihel's Portfolio (Next.js)

A modern, interactive developer portfolio built with **Next.js**, showcasing my skills, projects, and growth as a software engineering student.

---

## ⚙️ Tech Stack
- Next.js  
- React  
- Three.js  
- Framer Motion  
- Tailwind CSS  
- TypeScript  

---

## 🔋 Features
- Modern hero section with animations  
- Responsive Bento grid layout  
- Interactive 3D elements  
- Projects & skills showcase  
- Smooth Framer Motion animations  
- Fully responsive design  

---

## 🤸 Run Locally

```bash
git clone https://github.com/ibtihel11/StudentPortfolio.git
cd portfolio
npm install
npm run dev
